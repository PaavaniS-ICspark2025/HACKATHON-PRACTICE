# HACKATHON-PRACTICE

This repository contains a simple notes organizer website created during a hackathon practice session. The site includes multiple pages and allows users to add, view, and organize notes using local storage.

## Available Pages

- `index.html` - Home page with project overview
- `notes.html` - Interface for creating and viewing notes
- `about.html` - Information about the project
- `contact.html` - Contact details

## Running the Site

You can view the site by opening any of the HTML files in a web browser. For example:

```bash
# from the project root
xdg-open index.html
```

Notes are stored in your browser's local storage and persist between sessions until cleared.
